# FindUAM

## Proyecto Final – Avance funcional

FindUAM es una aplicación móvil desarrollada para la publicación, búsqueda y recuperación de objetos perdidos dentro de la Universidad Americana (UAM). La aplicación permite que los estudiantes puedan registrarse, iniciar sesión, buscar objetos, visualizar publicaciones recientes, publicar objetos encontrados y editar su perfil.

El objetivo principal del proyecto es centralizar la información de objetos perdidos dentro del campus, complementando el proceso actual de Objetos Perdidos en administración y facilitando la recuperación de pertenencias de una forma más rápida, organizada y accesible.


## Descripción del proyecto

Actualmente, cuando un estudiante pierde un objeto dentro de la universidad, normalmente debe preguntar en grupos de WhatsApp o acudir personalmente al área de administración. Aunque existe un espacio físico de Objetos Perdidos, el proceso sigue siendo manual y poco visible para los estudiantes.

FindUAM propone una solución digital mediante una aplicación móvil donde los usuarios pueden publicar objetos encontrados, buscar por nombre o categoría, ver publicaciones recientes y contactar a la persona que realizó la publicación.

## Funcionalidades implementadas

* Registro de usuario.
* Inicio de sesión.
* Visualización de objetos publicados.
* Búsqueda de objetos por nombre.
* Espacios de categorías para organizar los objetos.
* Publicación de objetos encontrados.
* Edición de perfil del usuario.

## Estructura general del sistema

El proyecto está dividido en dos partes principales:

### Aplicación Android

La aplicación móvil contiene las pantallas principales del sistema, modelos, navegación, componentes reutilizables y conexión con la API REST mediante Retrofit.

### API REST

El backend desarrollado en Spring Boot permite gestionar los datos principales del sistema, como usuarios y objetos publicados. Esta API se conecta a una base de datos PostgreSQL administrada desde pgAdmin.
